# QuickApp

- 个人中心

  - 修改个人信息

    ```json
    {
        "username":"",
        "password":"",
        "sex":"",
        "age":21
    }
    ```

  - 查看个人信息

    ```json
    {
        "username":"",
        "password":"",
        "sex":"",
        "age":21
    }
    ```

- 登录注册

  - 登录: 用户名、密码

    ```json
    {
        "username":"",
        "password":"",
    }
    ```

  - 注册：用户名、密码、性别、年龄

    ```json
    {
    	"username":"",
        "password":"",
        "sex":true,
        "age":20
    }
    ```
